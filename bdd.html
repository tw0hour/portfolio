Concepion de base de données

Dans beaucoup de mes projets scolaires, il était nécessaire pour les étudiants de créer des bases de données complètes afin de pouvoir réaliser ces projets correctement.

Une base de données est l’endroit dans lequel nous stockons des données. Les données sont stockées sous forme de tableaux ils sont organisés de manière à être facilement accessible, géré et mis à jour. C’est grâce aux bases de données que nous pouvons notamment nous inscrire sur des applications ou poster des commentaires etc.

Depuis la conception des MCD (Model conceptuelles de données) jusqu’au commande SQL (ou MongoDB) en passant par les MLD (Modèle logique de données) j’ai pu acquérir des notions importantes sur le monde de la conception de base de données, notamment sur le choix du type de BDD à choisir en fonction de ses besoins. Une base SQL est très utile pour stocker des données structurées, telles que des tableaux avec des colonnes prédéfinies, quant aux langages NoSQL, il est plus approprié pour stocker des données plus flexibles qui seront amené à évoluer.
