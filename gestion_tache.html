Application de gestion de tâches


Contexte du projet

Dans le cadre d'un projet scolaire, j'ai conçu une application de gestion de tâches en Java avec le framework JavaFX pour l'interface graphique. J'ai également créé une API en Node.js pour effectuer des requêtes vers ma base de données, que j'ai modélisée en SQL.


Fonctionnalités de l'application

Dans cette application, il est possible de créer plusieurs projets et plusieurs profils. Une fois qu'un projet est créé, il est possible d'ajouter des tâches. Lorsqu'une tâche est créée avec un titre, une description et un niveau de difficulté, elle est placée dans une colonne appelée "À faire" qui répertorie les tâches à réaliser. Il est possible de débuter une tâche en l'assignant à une personne, et une fois cela fait, la tâche est déplacée vers la colonne "En cours" qui répertorie les tâches en cours de réalisation. Enfin, lorsqu'une tâche est terminée, elle est déplacée vers la colonne "Terminé".

Les fonctionnalités supplémentaires de l'application comprennent :

  • - La possibilité d'annuler une tâche pour la replacer dans la colonne "À faire".

  • - La possibilité de modifier une tâche existante.

  • - La possibilité de supprimer une tâche.

  • - La possibilité de supprimer des utilisateurs.


Conclusion

Ce projet de gestion de tâches en Java avec une API Node.js pour la communication avec la base de données a été une expérience enrichissante dans mon parcours scolaire. Il m'a permis d'approfondir mes compétences en développement logiciel, en utilisant des technologies telles que JavaFX et Node.js. La création de cette application m'a également donné une meilleure compréhension de la gestion de projets et de la modélisation de bases de données.
