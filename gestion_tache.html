Application de gestion de tâches

Dans le cadre d’un projet scolaire, parmi les applications faites. J’ai conçu une application de gestion de taches en JAVA. (avec le framework javafx pour le graphique). Ainsi qu’une api en node js, que j’ai créée pour faire des requêtes dans ma base de donnée. (Que j’ai modélisée en SQL).

Dans cette application, il est possible de créer plusieurs projets, et plusieurs profils. Une fois un projet créée, il est possible de créer des taches. Quand une tache est créée (titre description, niveau de difficulté), elle se range dans une colonne nommé « to Do » cette colonne répertoriera les taches à faire. Il est possible de commencer en assignant une personne à celle-ci, une fois cela fait, la tache se rangera dans une colonne « in progress » qui répértorie les tâches en cours. Enfin quand une taches est terminé, elle sera rangé dans la colonne « finished ». il est également possible :


  • • D’annuler une tache pour que celle-ci repasse en « to do ».
  • • De modifier une tache
  • • De supprimer une tache.
  • • De supprimer des utilisateurs
